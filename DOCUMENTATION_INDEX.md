## 🎓 Design Patterns Used

### Visual Patterns
- **Glassmorphism**: Gradient + backdrop blur + ring effects
- **Card Pattern**: Self-contained information units with interaction
- **Micro-interactions**: Hover effects, scale transforms, shadows
- **Color Coding**: Status indicated by color (red/green/blue/amber)

### UX Patterns
- **Progressive Disclosure**: Collapsible alternative search
- **Feedback**: Loading states, error messages with icons
- **Affordance**: Hover effects show cards are interactive
- **Hierarchy**: Important info larger, secondary text smaller

### Code Patterns
- **Icon Components**: Reusable SVG icon functions
- **Helper Functions**: fmt(), normalizeCitizenStatus(), pick()
- **Responsive Classes**: Mobile-first with sm: and lg: prefixes
- **Accessibility**: Semantic HTML, ARIA-friendly structure
